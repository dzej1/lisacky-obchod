## Node modules installation

```bash
$ npm install
```

## .env setup

```bash
$ cp .env.example .env
```

## Run db migration

```bash
$ npx prisma migrate dev
```

## Running the app

```bash
# development
$ npm run start
```

## App URL

[App itself](http://localhost:3000/)
[OpenAPI (swagger)](http://localhost:3000/api/)
